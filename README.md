# Web Programming Course Portfolio

This repository showcases a collection of projects developed for a Web Programming course, demonstrating various web technologies and concepts.

---

## Projects Overview

### 1. Assignment 2: CSS Animations
**Description:** Exploration of CSS animations with a 3D cube and fade effects.
**Technologies:** HTML, CSS

### 2. Assignment 3: Course Preview Page
**Description:** A static web page designed as a Udemy course preview, focusing on layout and styling.
**Technologies:** HTML, CSS

### 3. Assignment 4: Cuphead Fan Page
**Description:** A dynamic fan page for the game Cuphead, featuring interactive elements and rich media.
**Technologies:** HTML, CSS, JavaScript

### 4. Assignment 5: TV Show Browser
**Description:** An interactive web page for browsing TV shows, demonstrating dynamic content display.
**Technologies:** HTML, CSS, JavaScript

### 5. Assignment 6: E-commerce Website
**Description:** A full-stack e-commerce application with a PHP backend and an Angular frontend.
**Technologies:** Angular, TypeScript, PHP, MySQL, HTML/CSS

### 6. DOOM Website
**Description:** A tribute website to the classic game DOOM, incorporating multimedia elements.
**Technologies:** HTML, CSS

### 7. Snake Game
**Description:** A classic Snake game implemented in Java, likely a desktop application.
**Technologies:** Java, Maven
